# BanksMS

**BanksMS** is a simple Bank Management System built with **ASP.NET Core MVC** and **Entity Framework Core**.  
It manages **Customers**, **Branches**, and **Accounts** in a SQL Server database.

---

## Features

- Add, update, and delete **Customers**  
- Manage **Branches** and their information  
- Create and manage **Accounts** with balance and account type  
- Connects to **SQL Server** using EF Core  
- Easy to extend and maintain  

---

## Technologies Used

- ASP.NET Core MVC  
- Entity Framework Core  
- SQL Server  
- C#
